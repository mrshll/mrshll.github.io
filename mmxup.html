mmxup

last updated 2021-04-22

Source code

mmxup entries have names, hosts, titles, and bodies. They optionally have dates. An entry can optionally be an index (examples: {wiki} and {mrshll.com}). The reference graph enables inbound and outbound relationships to be rendered.

The markup resembles markdown with a few key differences inspired by indental(for metadata) and kaku (for markup syntax), with unique design decisions based on my preferences.
